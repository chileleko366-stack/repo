# SFX Assets

Place Kenney SFX files here (CC0 — free download at kenney.nl/assets).

Required filenames (MP3):

| File | Used for |
|------|----------|
| `hit.mp3` | Beat-entrance thud (every beat, 2 frames before start) |
| `reveal.mp3` | Person asset entrance |
| `pop.mp3` | Brand / product / app logo entrance |
| `tick.mp3` | Stat counter ticking (every 6 frames, 45%–90% of beat) |
| `swoosh.mp3` | Map / distance beat entrance |
| `sting.mp3` | Twist section |
| `bell.mp3` | Outro (30 frames in) |

Recommended Kenney pack: **Interface Sounds** and **Sci-Fi Sounds**.
